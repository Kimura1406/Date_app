# Badge Concept Pack

Thu muc nay chua bo concept badge de duyet style truoc khi dua vao app/admin.

File:

- `badge-style-board.svg`: style board tong hop tat ca badge
- `index.html`: preview nhanh trong browser

Danh sach badge dang duyet:

- Rose: Blue, Red, Purple
- Diamond: 1ct, 5ct, 7ct, 10ct, 17ct
- Gold Ring: Plain, Diamond Halo
- Diamond Bouquet: 1, 2, 5, 10, 50, 99
- Luxury Ring: Green Gem, Red Gem

Neu chot style, buoc tiep theo se la:

1. Tach tung badge thanh asset rieng
2. Dinh nghia data model badge
3. Dua vao app/admin
